o

OPCIÓN: Visual Studio Code (Recomendado)
Instalar Extensión:
Abre VS Code
Ve a Extensions (Ctrl+Shift+X)
Busca "PlantUML" de jebbs
Instala la extensión
Usar:
Abre el archivo .puml
Presiona Alt+D para ver preview
O clic derecho → PlantUML: Preview Current Diagram
Para exportar: Clic derecho → PlantUML: Export Current Diagram
Requisitos:
Java instalado (JRE 8 o superior)
O GraphViz instalado
